# 🚀 AI Resume Analyzer

[![Live Demo](https://img.shields.io/badge/Live-Demo-green)](https://charankommireddy.github.io/Resume-analyzer/)

AI Resume Analyzer is a **web-based AI tool that evaluates resumes and provides detailed feedback** including resume score, ATS compatibility, keyword analysis, and improvement suggestions.

This project helps **students, fresh graduates, and job seekers improve their resumes** and increase their chances of passing automated recruitment systems.

# 📌 Project Description

Most companies use **Applicant Tracking Systems (ATS)** to filter resumes before they reach recruiters. Many resumes are rejected simply because they do not contain the right keywords or formatting.

This project analyzes resumes and provides insights such as:

* Resume score
* ATS compatibility
* Keyword analysis
* Skill detection
* Suggestions for improvement

The goal is to **help candidates create stronger resumes and improve their chances of getting shortlisted**.

# 🧠 How It Works

1️⃣ User uploads a resume or pastes resume text

2️⃣ The system extracts text from the document

3️⃣ The resume is analyzed using:

* ATS rules
* keyword detection
* skill analysis
* AI evaluation

4️⃣ The system generates:

* Resume Score
* Strengths
* Improvement Suggestions
* Skill Analysis
* ATS Compatibility Report

5️⃣ Results are displayed in an **interactive dashboard**

# ⚙️ Installation

Navigate to the project folder

cd Resume-analyzer

Install dependencies

npm install

Run the server

npm start

Open in browser

http://localhost:3001
